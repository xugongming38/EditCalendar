# EditCalendar
这是一个 Android 平台上可标记的自定义日历控件。
非完全原创 项目，基于github上另外一个自定义日历控件上加入了数据库的相关操作，完成点击以及撤销操作
基本 完成自定义标记，欢迎二次开发，欢迎star
